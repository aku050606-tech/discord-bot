# HP持ち越し・アナウンス廃止 修正メモ

## 海の白兵戦HP

- 海の白兵戦開始時、現在HPを引き継ぐように変更。
- 戦闘終了時・撤退時に、戦闘中の残HPを `voyage.cur_hp` へ保存。
- 出航・帰港だけでなく、戦闘突入でもHPが全回復してしまう挙動を防止。
- 包帯使用時も現在HPを即時保存。

## 大勝利アナウンス

- 調整が煩雑になっていたため、一旦完全廃止。
- `cogs.bigwin.announce_big_win()` は no-op になり、呼ばれても投稿しない。
- 各ゲーム側の呼び出しは残しているため、将来復活させる場合は `announce_big_win()` の中身を戻せばよい。

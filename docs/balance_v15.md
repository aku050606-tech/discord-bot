# Balance v15

## 目的
- 海洋はコイン稼ぎ、街道は経験値・探索体験の軸に分離。
- 既存のイベント/戦闘頻度は大きく変えず、各エリアの難易度だけを段階化。

## 基準
- 🌊 海洋 E2：Lv20 + ☆2フル装備でギリギリ進める想定。
- 🌊 海洋 E3：Lv30 + ☆3フル装備で通常敵を処理できる想定。強敵/ボスは危険枠。
- 🌲 森：Lv15 + ☆1フル装備で通常敵を処理できる想定。中レアは強敵。
- ⛰️ 山：Lv25 + ☆2フル装備で通常敵を処理できる想定。中レアは強敵。

## 方針
- 全体の出現率テーブルは大きく変更しない。
- 強敵扱いの敵は平均より明確に強くする。
- 熱いイベントは通常イベント枠の中で低確率出現。出た場合はアイテム/コイン/戦闘などの振れ幅を大きくする。

# 変更メモ：アナウンス抑制・装備作成コスト調整

## アナウンス

大勝利アナウンスが頻繁に流れすぎていたため、閾値を上げました。

- 航海: 100,000 → 300,000
- スロット: 30,000 → 100,000
- 釣り/宝の地図: 50,000 → 150,000

通常の良い結果では告知せず、本当に大きい勝ちだけを告知します。

## 装備作成コスト

現状のクラフト素材要求が重すぎたため、必要素材を軽量化しました。

- ☆2装備: 必要素材を約40%軽量化
- ☆3装備: 必要素材を約45%軽量化

装備作成の導線はそのまま維持しつつ、素材集めの詰まりを減らす調整です。

# 街道イベント土台 v9

## 方針
イベントは「結果」ではなく「きっかけ」。

1. メイン抽選
   - 戦闘 / 中ボス / 大ボス / ストーリー / ランダムイベント / アイテム / コイン / 採取 / 何もなし
2. イベント抽選
   - エリアごとのイベントを選ぶ
3. 選択肢
   - プレイヤーが選ぶ
4. サブ抽選
   - コイン / アイテム / 戦闘 / ダメージ / 回復 / XP / 世界観テキスト / 何もなし

同じイベントでも毎回結果が変わるため、探索の単調さを減らす。

## 追加イベント数
- 平原 +10
- 森 +8
- 山 +8

既存イベントと合わせて、各エリアのイベント密度を上げた。

## 追加場所
`land_config.py` の `LAND_RANDOM_EVENT_PACK_V9`。
コード本体ではなく、データ追加で拡張できる形。

# 海イベント リスク調整メモ

## 変更方針

- 船体HPイベントは廃止。
- 船への負担はすべて燃料減少で表現する。
- 報酬がある選択肢には、原則として燃料/HP/戦闘などのリスクを付ける。
- カルマが下がる悪行選択は、報酬を増やす代わりに燃料/HPリスクも重くする。
- 見送る/離れる/何もしない系の安全選択肢は残す。

## 具体調整

- `ship_hp` は読み込み時の後処理で `fuel` に変換。
- 悪行かつ報酬ありの選択肢は報酬を約1.35倍。
- 悪行選択肢には追加で燃料消費とHP消耗を付与。
- 素材/宝/釣り/カケラなどの報酬選択肢にも燃料消費を付与。
- 古い `ship_hp` データが残っても、実行時に燃料変動へ変換する安全処理を追加。

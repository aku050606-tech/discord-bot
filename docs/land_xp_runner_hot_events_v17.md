# Land v17: XP Runner & Hot Event Drama

## 追加内容

### 💎 経験値逃走モンスター
各エリアの戦闘枠内に低確率で出現する経験値専用モンスター。

- 通常版：約1.0%
- キング版：約0.1%
- 毎ターン50%で逃走
- ドロップなし（装備・アイテム平均ドロップ率は変更しない）
- 通常版：通常雑魚XPの約20倍
- キング版：通常雑魚XPの約100倍

### 🔥 熱いイベント
既存の熱いイベントの出現率は維持しつつ、リスク/リターンを約5倍寄せ。

- coin / xp / heal / damage の効果量に倍率
- item は出現率を変えず、当たった時の中身だけ夢寄り
- combat は中ボス寄りに変更
- 前兆テキストを追加

## 注意
